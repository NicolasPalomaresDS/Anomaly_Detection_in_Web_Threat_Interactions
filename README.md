# Cybersecurity: Anomaly Detection in Suspicious Web Threat Interactions Using Machine Learning

The present project is aimed at detecting suspicious activities and potential attack attempts in web traffic interactions. It follows a systematic approach:

> **Baseline Model**: A simple statistical detector using the Interquartile Range (IQR) method establishes the minimum performance threshold. This baseline identifies extreme outliers based on traffic volume (bytes transferred).

> **Advanced Model**: An Isolation Forest algorithm captures multidimensional patterns that the baseline cannot detect, considering multiple features simultaneously (traffic volume, connection duration, time of day, geographic origin).

> **Comparative Analysis**: Both models are evaluated and compared using clustering quality metrics (Silhouette Score) to assess separation between normal and critical threats.

The data was generated by monitoring traffic to a production web server, using various detection rules to identify anomalous patterns.

### Directory Structure
* `notebook.ipynb`: Jupyter Notebook with the code.
* `requirements.txt`: Required libraries for notebook execution.
* `scripts/`: Custom Python scripts.
    - `scripts/kaggle_setup.py`: For data download and Kaggle configuration.
    - `scripts/df_summary.py`: For a complete dataset summary generation.
    - `scripts/baseline_model.py`: Script with baseline model class.
